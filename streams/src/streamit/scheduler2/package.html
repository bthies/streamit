]> Overview of streamit.scheduler2.iriter

The main scheduler2 package. Here you find all the important classes the provide an interface between the scheduler and the user.

Schedule class implements a basic phasing schedule.

Scheduler provides an interface between the user and the scheduler. Each scheduler implementation must provide its own Scheduler subclass.

SDEPData is a basic interface for represending SDEP relationship between two filters.

This package also contains the ScheduleOptimizer and ScheduleBuffers algorithms. ScheduleOptimizer performs basic factoring of a schedule, based on repetitions of phases and single-element phases. ScheduleBuffers computes the size of buffers necessary to execute a given schedule.
